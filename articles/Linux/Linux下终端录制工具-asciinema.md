## 官网

[https://asciinema.org/](https://asciinema.org/)

## 功能

- 录音
- 录制终端命令
- 上传到多种位置

## 使用方法

``` shell
sudo yum install asciinema
# 安装

asciinema auth
# 进行自动配置，打开给你的链接，进行简单的注册，

asciinema rec
# 开启录制，你的录制内容，将会自动上传到他的官方网站

# Ctrl+D或exit是终止录制
```
上传成功后，会给你一个播放的链接。录制的内容默认是私有的，不是公开的。
当你运行录制的命令开始，就已经开始录制了。
十分清晰，很给力。
